# Add base Facebook colors

Primary/Secondary Facebook colors: BLUE_MARKETING, BLUE_FIG,
BLUE_HIGHLIGHT, TEAL, LIME, LEMON, ORANGE

## Usage

``` r
add_colors_facebook_y2(palette = 1, show_colors = TRUE)
```

## Arguments

- palette:

  DEFAULT = 1. Defaults to the newest facebook color palette (1).

- show_colors:

  DEFAULT = TRUE Show the color palettes loaded into your environment

## Details

Adds to your environment pre-set hex codes for the Facebook blues along
with 4 secondary colors used in Facebook themed projects. There are an
additional 11 tertiary colors. The 4 blues have 7 shades. Also adds
BLACK, WHITE, and GRAY, while optionally displaying the colors/hex codes
for you to see.

## Examples

``` r
add_colors_facebook_y2()

add_colors_facebook_y2(show_colors = TRUE)
```
